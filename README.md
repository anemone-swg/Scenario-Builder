## 🚀 Запуск проекта

### 1. Клонирование репозитория
<pre>
git clone https://github.com/anemone-swg/Scenario-Builder.git
cd Scenario-Builder
</pre> 

### 2. Установка зависимостей
npm install

### 3. Запуск в режиме разработки
npm run dev

### 4. Сборка для production
npm run build

## 🏗 Архитектурные решения

### 1. **Feature-Sliced Design (FSD)**
Проект организован по методологии Feature-Sliced Design, 
<pre>
src/
  ├── app/ # Инициализация приложения (роутинг, store)
  ├── pages/ # Страницы приложения
  ├── widgets/ # Виджеты приложения
  ├── features/ # Сценарии использования (фичи)
  ├── entities/ # Бизнес-сущности (Scenario, Node, Edge)
  ├── shared/ # Переиспользуемые компоненты, утилиты
</pre>
